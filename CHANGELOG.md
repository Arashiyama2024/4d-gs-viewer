# Changelog

このプロジェクトのすべての変更を記録します。

## [0.3.0] - 2025-XX-XX

### 追加
- シーン環境切替機能（デフォルト / 3Dトラッキングスタジオ / クロマキー配信用）
- 3Dトラッキングスタジオ: グリーンバック + 58個のトラッキングマーカー（被写体回避配置）
- クロマキー: 完全プレーンなグリーン背景（OBSクロマキーフィルタ用）
- データ読み込みタブ復元（ファイル / URL / サンプルデータ / 対応形式 / 参考リポジトリ）

## [0.2.0] - 2025-XX-XX

### 追加
- フリーズ機能（一時停止でポーズ完全固定、カメラのみ操作可能）
- ローアングル対応（仰角 -0.8 まで地面下へ）
- CSS ベースのフルスクリーン（iframe 環境対応）
- マウス操作（左ドラッグ回転、ホイールズーム、右ドラッグ仰角）

### 変更
- 自動 Orbit カメラ回転を削除（VFX 合成向け）
- カメラモデルをルックアット方式に変更（被写体が常にセンター）

## [0.1.0] - 2025-XX-XX

### 追加
- 初版リリース
- 人型デモモデル（約 4,200 ガウシアンスプラット、ダンスアニメーション）
- マルチカメラプリセット管理（6 プリセット、キーボード 1〜6 即切替）
- ディレクターパネル（カメラ切替 / トランジション / 微調整スライダー / 操作ログ）
- 配信設定（OBS / YouTube / Zoom / Twitch プリセット、OBS 設定 JSON コピー）
- 日本語 UI
- 4D Gaussian Splatting 解説ツールチップ
